# prisma-ffi 参考

C FFI 共享库，用于 GUI 和移动端。

## 错误码

PRISMA_OK(0)、PRISMA_ERR_INVALID_CONFIG(1)、PRISMA_ERR_ALREADY_CONNECTED(2)、PRISMA_ERR_NOT_CONNECTED(3)、PRISMA_ERR_PERMISSION_DENIED(4)、PRISMA_ERR_INTERNAL(5)、PRISMA_ERR_NULL_POINTER(6)

## 导出函数

生命周期、连接控制、配置文件、QR 码、URI 导入、系统代理、自动更新、分应用代理、代理组、端口转发、移动端生命周期。

详细内容请参阅英文版本。
