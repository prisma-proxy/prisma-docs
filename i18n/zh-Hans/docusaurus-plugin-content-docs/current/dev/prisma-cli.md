# prisma-cli 参考

CLI 二进制，使用 clap 4 构建。

## 命令

server、client、console、gen-key、gen-cert、init、validate、status、version、completions、import、clients、connections、metrics、bandwidth、config、routes、logs、ping、diagnose、speed-test、subscription、latency-test

## 守护进程管理

所有服务支持 -d 后台运行、stop 停止、status 检查状态。

详细内容请参阅英文版本。
